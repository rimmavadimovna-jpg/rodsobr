# Источники для наполнения банка

Сюда кладутся исходные материалы для выверки банка (не распространяются вместе
с ботом, если охраняются авторским правом).

- `Перечень_слов_безударные_гласные.docx` — перечень слов для зад. 1
  (безударные/чередующиеся гласные). Парсится в таблицу `words`
  (см. `build_bank.seed_words` — расширь под свой формат `.docx`).

Справочники (источник для наполнения и выверки, **не** для онлайн-запросов):

- Розенталь — https://old-rozental.ru/
- Лопатин (Правила-2006) — https://orthographia.ru/
- Грамота.ру (словари) — https://gramota.ru/
- Викисловарь — https://ru.wiktionary.org/ (обязателен для зад. 9)

У каждой единицы банка — поле `source` со ссылкой на конкретный
параграф/статью.
